\doxysubsection{Detailed Description}
Fee model class.

This class represents a fee entity and provides methods to interact with the {\ttfamily fees} table in the database. It includes fields like CIF, concept, issue date, amount, passed status, payment date, and notes. The {\ttfamily client} relationship links the fee to a client via the CIF field.
